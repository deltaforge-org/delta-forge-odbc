DeltaForge ODBC Driver 1.0.6.

Native ODBC driver for Delta Lake: point Power BI, Tableau, Excel, .NET,
Python, and R at your tables through the standard Driver Manager. Full read
and write, plus a resident catalog cache for fast metadata.

Includes: driver shared library (Linux .so, macOS .dylib, Windows .dll),
platform installers (.deb, .rpm, .pkg, .msi), and BI tool connector
skeletons (Tableau .taco, Power Query .pqx, Looker JDBC bridge, Qlik
DAG, MicroStrategy VLDB profile, Apache Arrow ADBC bridge).

See SETUP_NOTES.md in each installer for registration steps.

More at https://deltaforge.org/pages/odbc.html
